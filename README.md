# Dhan CLI Dist

Public release artifacts and agent skill for Techriff.DhanCli (`dhan`).

## Install the CLI

See https://cli.jorli.in/install/

## Install the agent skill

```bash
npx skills add ssswagatss/dhan-cli-dist --skill dhan-cli
```

Global (all agents):

```bash
npx skills add ssswagatss/dhan-cli-dist --skill dhan-cli -g --agent '*' -y
```

Source of truth: private `ssswagatss/dhan-cli` → `skills/dhan-cli/` (synced on push to main).
