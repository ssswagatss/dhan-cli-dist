# dhan edis

Electronic Delivery Instruction Slip flows (T-PIN / bulk auth).

```bash
dhan edis tpin --json --no-color
dhan edis form --file edis.json --json --no-color
dhan edis inquire --isin <isin> --json --no-color
```

Check exact subcommands with `dhan edis --help`. Do not invent ISINs or quantities.
