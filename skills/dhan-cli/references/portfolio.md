# dhan portfolio

```bash
dhan portfolio holdings --json --no-color
dhan portfolio positions --json --no-color
dhan portfolio convert --file ./convert.json --yes --json --no-color   # LIVE + static IP
dhan portfolio exit-all --yes --json --no-color                        # LIVE + static IP
```

Prefer dry-run if your binary supports it on convert (`--help`). `exit-all` is destructive — only with explicit operator instruction.
