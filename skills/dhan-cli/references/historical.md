# dhan historical

```bash
dhan historical daily --security-id <id> --segment NSE_EQ --from <d> --to <d> --json --no-color
dhan historical intraday --security-id <id> --segment NSE_EQ ... --json --no-color
dhan historical rolling-options ... --json --no-color
```

Exact flags vary — always run `dhan historical --help` and leaf `--help`. Use numeric `securityId` + segment.
