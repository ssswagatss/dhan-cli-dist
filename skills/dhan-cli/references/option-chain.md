# dhan option-chain

Requires Dhan Data API.

```bash
dhan option-chain expiry-list --underlying-scrip <id> --segment NSE_FNO --json --no-color
dhan option-chain get --underlying-scrip <id> --segment NSE_FNO --expiry <date> --json --no-color
```

Verify flag names with `dhan option-chain --help` on the installed binary.
